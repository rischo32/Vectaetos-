## Reporting vulnerabilities

If you discover a security-related issue affecting:
- the repository integrity
- deployment infrastructure
- website behavior

please report it privately via GitHub Security Advisories.

Public disclosure without coordination is discouraged.

## Scope

This policy applies to:
- repository infrastructure
- deployment workflows
- public website hosting

It does NOT apply to:
- philosophical disagreement
- theoretical critique
- alternative interpretations of the ontology
