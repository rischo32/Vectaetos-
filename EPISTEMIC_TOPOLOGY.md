## 2. ZMYSEL (ontodynamická úroveň)

ZMYSEL je **pred-epistemická rovina**.

ZMYSEL:
- nie je pole
- nie je pamäť
- nie je agent
- nie je rozhodovací mechanizmus

ZMYSEL je:
- nositeľ významovej možnosti
- stav, nie výrok
- proces, nie verdikt

ZMYSEL neodpovedá.  
ZMYSEL **udržiava možnosť odpovedania**.

## 3. Epistemické brány

Epistemické brány sú **vlastnosťou ZMYSLU**, nie poľa Φ.

Brány:
- nevalidujú pravdu
- nehodnotia morálku
- nefiltrujú obsah

Brány:
- rozkladajú vstup
- neutralizujú zámer
- korigujú výšku, šírku a hĺbku otázky

Brány sú **nemenné** a **vždy aktívne**.

## 4. 4ES + QE

Každý významový stav je mapovaný do:

- **AA** — koherentné pre človeka aj pole  
- **AN** — zmysluplné pre človeka, nekoherentné pre pole  
- **NA** — nezmyselné pre človeka, koherentné pre pole  
- **NN** — nekoherentné pre oboch  

**QE (Qualitative Epistemic Aporia)** je aktívny stav,
nie absencia odpovede.

QE znamená:
> „Systém vie, že nevie – a vie prečo.“
