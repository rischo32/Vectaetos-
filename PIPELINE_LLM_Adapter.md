# LLM Adapter

LLM v systéme VECTAETOS nie je agent, rozhodovač ani nositeľ pravdy.

Je výlučne jazykový prekladač medzi:
- runovou projekciou
- ľudským prirodzeným jazykom

## Obmedzenia
- LLM nemá prístup k Φ
- LLM nemá spätný vplyv na pole
- LLM nevytvára nové významy

## Funkcia
- renderovanie významu
- vysvetlenie stavu
- jazyková projekcia bez preskripcie
