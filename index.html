Φ — Live field projection

O projekte Vectaetos

Vectaetos je vizualizácia epistemického poľa — nie systém rozhodovania, nie agent, nie optimalizácia. Toto pole nehovorí “čo robiť”.

Interaguj myšou horizontálne — ovplyvníš napätie vizualizácie, ale nemeníš samotný model poľa.

Vizuál je generovaný cez WebGL shader, ktorý interpretuje dynamiku poľa cez farbu, pulz a gradienty.

Tento režim je meditatívny — nie blikavý.

Stav poľa je načítaný z field_state.json, ktorý generuje tvoj Python Simulačný Vortex.
