# VECTAETOS — WEB ARTEFACT
Status: Canonical
Scope: Definition of the public web interface as epistemic projection
Immutability: Ontological meaning fixed

---

## 1. Ontologický status webu

Web VECTAETOSU nie je aplikácia.
Nie je rozhranie k systému.
Nie je interakčný agent.

Web je **epistemický artefakt**:
projekcia vzťahov v poli Φ do vizuálnej formy.

Web:
- nevykonáva výpočty významu
- nerozhoduje
- neodpovedá
- nepamätá si
- neučí sa

---

## 2. Čo web robí

Web:
- vizualizuje axiomatické ťažiská Σ₁…Σ₈
- zobrazuje tenzie medzi nimi
- umožňuje jednorazový epistemický vstup
- projekčne ukazuje vzťah, nie riešenie

Výstup webu je:
**zvýraznený vzťah**, nie odpoveď.

---

## 3. Čo web nikdy nerobí

Web:
- ❌ neposkytuje rady
- ❌ nenavrhuje kroky
- ❌ negeneruje odpovede
- ❌ nehodnotí používateľa
- ❌ neukladá vstupy
- ❌ neporovnáva session
- ❌ neobsahuje spätnú väzbu

Ak by niektorá z týchto vlastností vznikla,
web by prestal byť súčasťou VECTAETOSU.

---

## 4. Vzťah k epistemickým bránam

Web:
- neimplementuje epistemické brány
- iba **vizualizuje ich prejav**

Sekvencia 3Gate:
- nie je validácia vstupu
- je projekcia epistemickej korekcie

Používateľ nevidí rozhodnutie,
vidí **zmenu poľa**.

---

## 5. Vzťah k Vortexu

Web:
- nie je riadený Vortexom
- Vortex môže byť zdrojom dát
- nikdy nie autoritou významu

Ak Vortex neexistuje:
- web zostáva funkčný
- pole je latentné
- projekcia je tichá

---

## 6. Ticho ako výstup

Ticho je plnohodnotný výstup.

Ak:
- neexistuje realizovateľná tenzia
- alebo by projekcia porušila K(Φ)

potom:
- web nič „neukáže“
- nevzniká chyba
- vzniká epistemická apória

---

## 7. Žiadna identita používateľa

Web:
- nepomenúva používateľa
- nevytvára profil
- nepozná históriu

Každý vstup je:
- izolovaný
- jednorazový
- ontologicky anonymný

---

## 8. Záver (kanonická veta)

Web VECTAETOSU nie je rozhraním k poznaniu,
ale zrkadlom napätia,
ktoré vzniká, keď poznanie naráža na vlastné hranice.
